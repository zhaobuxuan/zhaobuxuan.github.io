Python语法初识

发表于 2018-09-18 | 分类于 Python

本文简要介绍Python语法，不涉及高深的东西，适合快速入门

入门推荐：唐宇迪 https://www.bilibili.com/video/av22404277/

几种常用类型：int float str bool

type(变量名) 打印出类型
tang =1.5 转换类型 tang2=int(tang) 值是1
type(tang2) int
str(tang) 转字符串

字符串操作

乘法 乘一个数 tang_str=”h “ tang_str*3=”h h h “
len(tang_str) 打印字符串长度
tang_str.split() 切分字符串，默认空格 可指定分隔符
.join(tang_str) 合并在一起
.replace(‘’,’’)
.upper() .lower() 大小写
.strip() 去掉空格 .lstrip() .rstrip()
‘{} {} {}’.format(‘’,’’,’’) 格式化 {}可定义存放位置 也可以指定参数
注意：对字符串修改后定义新的变量

List

通过[]来创建一个list结构,里面放任何类型都可以，没有一个长度限制
tang=[1,2,3]
tang=list([1,2,3])

操作：
len(tang) 长度
两个list想加，合并list
乘一个数，复制list并合并
索引基0，tang(0:) 从零开始打印所有
通过索引找到位置，可替换对应元素。 a[:]=[‘’,’’] 替换所有
删除del tang[0] 则 tang=[2，3]
del a[3:] 从3开始全部删除
8 in a not in 也可以判断字符串串 true false

核心操作：
list可嵌套list
.count(‘’) 括号内传入需要计数的参数 可统计list中有多少个此种参数
.index(‘’) ——- 定位
.append(‘’) 每次只能在最后增加一个元素，可添加list结构
.insert(2,’’) 在第二个位置插入某个东西
.remove(‘’) 可去掉list，如果有多个元素，会去掉第一个
.pop(1) 弹出第一个元素 原list中消失
.sort() 排序 无参数
tang2=sorted(tang) 此种方法原始数据不变 tang2是新数据
.reverse() 反序

字典dict {}定义 key-value

字典的结构操作
tang={} tang[‘first’]=123
键是唯一的
取值 tang[‘first’] 或者.get(‘’)
字典中没有顺序之分
核心操作：
key一般用字符串 value可以存放list、字典，可以任意存放

可以对键值进行操作
.pop(‘键值’) 弹出操作，弹出value
del tang[‘键值’] 删除对应键值对
原字典.update(新的字典) 对原字典进行更新操作 若key重复，则value以新字典为准
‘tang’ in tang 判断键在不在字典当中
.keys() 打印所有的键
.values() 打印所有值
.items() 都打印

集合set 集合会保留下来唯一的元素

tang=set([123,123,456,456])
123 456
set即对list去重
或者tang={11,11,22,22} 11 22

集合的操作：
交集 a={1,2,3,4} b={2,3,4,5} a.union(b)或者a|b {1,2,3,4,5}
并集 a.intersection(b) 或者 a&b {2,3,4}
差异 a.difference(b)相当于a-b b.difference(a)相当于b-a
b.issubset(a)或者b<=a b是不是a的子集
.add(值) 添加一个值
.update(参数)
.remove(参数)
.pop() 弹出最前面的值
