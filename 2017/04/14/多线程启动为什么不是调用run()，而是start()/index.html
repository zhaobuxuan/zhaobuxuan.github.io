多线程

发表于 2017-04-14 | 分类于 多线程

此处查看JDK源码（jdk安装目录下——src.zip中——java.lang——Thread.java）

在start方法中存在 IllegalThreadStateException()异常抛出，
此异常属于java.lang.RuntimeException子类

IllegalThreadStateException - if the thread was already started.
如果线程被重复启动，则抛出。

==重点==

Start() 方法调用 start0() ,start0此方法的结构与抽象方法类似

Native声明，在java开发技术中有一门叫JNI技术（java native interface）

特点：使用java调用本机操作系统提供的函数。

缺点：不能离开特定的操作系统

如果要线程能够执行，需要操作系统进行资源分配。所以此操作严格来讲是根据JVM根据不同的操作系统实现的

即：==使用Thread类的start()方法不仅要启动多线程的执行代码，还要不同的操作系统进行资源的分配==
